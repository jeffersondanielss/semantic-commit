# Semantic Commit cli

![how use preview](preview.gif)

## Install

Clone o repositório
```bash
  $ git clone git@github.com:jeffersondanielss/semantic-commit.git
```

Entre na pasta do repositório
```bash
  $ cd semantic-commit
```

Instale globalmente

```bash
  $ npm i -g
```
