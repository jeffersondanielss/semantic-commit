# Semantic Commit cli

![how use preview](preview.gif)

## Install

Clone the repository
```bash
$ git clone git@github.com:jeffersondanielss/semantic-commit.git
```

Enter the repository folder
```bash
$ cd semantic-commit
```

Install Globally
```bash
$ npm i -g
```
